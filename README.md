# PDF to Word Converter using Python

## Description
This project recreates a given PDF document into an MS Word document using Python.
The document structure, spacing, headings, and boxes are recreated using python-docx.

## Tools & Technologies
- Python 3.x
- python-docx

## Approach
- Studied the PDF layout and structure carefully
- Recreated the document manually using python-docx
- Used tables to represent boxes
- Applied fonts, alignment, spacing, and formatting programmatically

## Output
- django_assignment.docx
